Alert
The Alert component is used to displays a short, important message in a way that attracts the user's attention without interrupting the user's task.
There can be different tyes of alerts depending on the nature it could be a dangerous warning or task succesfully executed alert.
Alerts can be created using the class 'alert' inside the div tag and also use the classnames 'alert-success', 'alert-primary', 'alert-danger' and 'alert-warning' for the different types of alerts.

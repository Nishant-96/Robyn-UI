Avatar
The Avatar component is used to represent a user, and displays the profile picture, initials or fallback icon.
Profile Avatar
Profile Avatars can be created using the class 'avatar', inside the div tag and also use the classes 'avatar-image avatar-circle' inside the img tag.
                                           

     <div class="avatar"><img src="../images/avatar10.jpg" alt="avatar"
     class="avatar-image avatar-circle" /></div>
Square Avatar
Rounded Square Avatars can be created using the class 'avatar', inside the div tag and also use the classes 'avatar-image avatar-square' inside the img tag.
                        

     <div class="avatar"><img src="../images/avatar10.jpg" alt="avatar"
     class="avatar-image avatar-square" /></div>
Sized Avatar
Size of the avatars can be changed by adding avatar-sml, avatar-mdm or avatar-lrge class to the existing class in the img tag as shown in code snippet. 
                        
            <!-- class "avatar-sml" for small size -->

     <div class="avatar"><img src="../images/avatar10.jpg" alt="avatar"
     class="avatar-image avatar-circle avatar-sml" /></div>

            <!-- class "avatar-mdm" for medium size -->
     
     <div class="avatar"><img src="../images/avatar10.jpg" alt="avatar"
     class="avatar-image avatar-circle avatar-sml" /></div>

            <!-- class "avatar-lrge" for large size -->
            
     <div class="avatar"><img src="../images/avatar10.jpg" alt="avatar"
     class="avatar-image avatar-circle avatar-sml" /></div>
